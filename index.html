Token = base64url(JSON) + . + checksum. The JSON carries full-name keys (funnel, linkLocation, variant, date, customTag) so it decodes self-describing, with every key always present (empty string when unset). Link location, variant and custom tag are optional. Date stamps automatically as ISO YYYY-MM-DD (today). The checksum lets Supabase detect a lead editing the value. Calendly drops non-UTM keys, so the code rides utm_content; it appears in Calendly's webhook payload.
